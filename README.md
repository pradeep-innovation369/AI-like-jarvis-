You can install all the required Python libraries using pip. Open Command Prompt / Terminal and run this command:
pip install pyttsx3 requests pywhatkit SpeechRecognition pyautogui googletrans==4.0.0-rc1 wikipedia langdetect psutil googlesearch-python pyaudio
⚠ Important Notes

speech_recognition needs pyaudio for microphone input.

On Windows, installing pyaudio may fail with pip. If it fails, run:
pip install pipwin
pipwin install pyaudio
Libraries this installs:

pyttsx3 → Text-to-speech engine
requests → HTTP requests
pywhatkit → WhatsApp automation, YouTube search
SpeechRecognition → Voice recognition
pyautogui → Keyboard & mouse automation
googletrans → Language translation
wikipedia → Wikipedia search
langdetect → Language detection
psutil → System monitoring
googlesearch-python → Google search from Python
pyaudio → Microphone access
